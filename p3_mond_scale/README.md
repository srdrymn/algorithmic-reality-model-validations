1. **Identify the noise floor:** The cosmological event horizon $R_E$ — the observer's ultimate causal boundary — generates an irreducible thermal noise at the Gibbons-Hawking temperature $T_{GH} = \hbar c / (2\pi k_B R_E)$.

2. **Identify the signal:** A gravitationally accelerating mass element generates a local Unruh temperature $T_U = \hbar a / (2\pi k_B c)$.

3. **Equate the scales:** When $T_U < T_{GH}$, the local gravitational signal is thermodynamically unresolvable. The threshold is $a_0 = c^2/R_E = cH_\infty$ — the MOND acceleration scale, derived from cosmological constants alone.
